//Comandos Básicos de Git//
git --version
Definición: Muestra la versión de Git instalada en el computador.
Ejemplo: git --version

git init
Definición: Inicializa un repositorio Git en la carpeta actual.
Ejemplo: git init

git add
Definición: Agrega archivos o cambios al área de preparación 
(staging).
Ejemplo: git add .

git commit
Definición: Guarda los cambios realizados en el repositorio con 
un mensaje descriptivo.
Ejemplo: git commit -m "Primer commit"

git status
Definición: Muestra el estado actual del repositorio y los 
archivos modificados.
Ejemplo: git status
